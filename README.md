# What is this?

My first project which displays if a number given is between 1 and 10.

# Installation

`npm i miraysfirstproject --save`

Then...

...
import { miraysfirstproject } from 'miraysfirstproject';

miraysfirstproject({
number: 5
});
...

# Options

- Numbers less than 1
- Numbers between 1 and 10
